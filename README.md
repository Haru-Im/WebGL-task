## 💡 프로젝트에 대하여

이 프로젝트는 WebGL, babylonjs, TypeScript 그리고 CRA(Create React App)로 개발되었습니다.

<br/>

## 💡 설치 및 실행 방법

1. 프로젝트 종속성 설치:
```bash
yarn install
```
<br/>

2. 💡 로컬에서 앱 실행:
```bash
yarn start
```
<br/>

이후 브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어서 프로젝트를 확인할 수 있습니다!
